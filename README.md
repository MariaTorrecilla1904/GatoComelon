#  Gato Comelon

Proyecto académico en Unity donde se implementan mecánicas básicas de un videojuego 3D.

## Funcionalidades

* Movimiento del jugador
* Spawner de objetos (comida)
* Uso de prefabs
* Sistema de colisiones
* Contador de puntos (UI)

##  Ejecución

1. Abrir en Unity Hub
2. Cargar la escena
3. Presionar **Play**

## Tecnologías

* Unity
* C#
